ABOUT



Park N Go is a hybrid mobile application which allows the user to search for the owner of the vehicle by entering the registration number of that vehicle. Park N Go has been developed on Eclipse IDE using IBM Worklight technogoly. Park N Go is structured by using HTML and CSS and the back-end programming is exhibited by JavaScript. The database connectivity is established using Firebase. This app provides a feasible solution to the parking misconduct across the country.
